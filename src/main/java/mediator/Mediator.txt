Шаблон Посредник

Цель: Инкапсуляция способа взаимодействия множества объектов.

Для чего используется: Для определения объекта, который мнкапсулирует способ взаимодействия множества
объектов и обеспечения слабой связи между этими объектами.

Пример использования: - связи между объектами сложны и четко определены;
                      - нельзя повторно использаовать объект, т.к он обменивается имформацией с другими оюъектами;
                      - поведение, распределенное между несколькими классами  должно легко настраиваться без
                        создания подклассов.
