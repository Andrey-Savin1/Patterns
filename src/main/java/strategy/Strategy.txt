Шаблон: Стратегия

Цель: Взаимозаменяемость семейств класса.

Для чего используется: Определение семейства классов, инкапсулирование каждого из них и организация их взаимодействия.

Пример: - в классе есть данные о которых не должен знать клиент;
        - необходимо иметь несколько вариантов поведения.
